To enable wsl on windows should enable ***windows subsystem linux*** in **windows feature**.

In **PowerShell** we can easily install any *linux distribution*:
```
>> wsl --version
>> wsl -- list

>> wsl --install
>> wsl --install -d Ubuntu

>> wsl --set-default <Name>
```
